# DEQJAM - 모바일 지원 2D 오픈월드 게임

DEQJAM은 2D 사이드뷰 오픈월드 게임으로, 모바일 터치 컨트롤을 완전히 지원합니다.

## 주요 기능

- 🎮 **모바일 터치 컨트롤**: 가상 조이스틱과 터치 버튼으로 완전한 모바일 지원
- 🌍 **오픈월드**: 무한 생성되는 2D 사이드뷰 월드
- ⛏️ **채굴 및 건설**: 블록을 채굴하고 건설하여 구조물 만들기
- 🛠️ **제작 시스템**: 다양한 아이템 제작
- 👾 **적과 전투**: 밤에 나타나는 좀비와 전투
- 💬 **NPC 상호작용**: PIKU와 거래 및 대화
- 📦 **인벤토리 시스템**: 아이템 관리 및 핫바

## 모바일 컨트롤

- **가상 조이스틱** (왼쪽 하단): 좌우 이동
- **점프 버튼** (오른쪽 하단): 점프
- **채굴 버튼**: 블록 채굴
- **설치 버튼**: 블록 설치
- **인벤토리 버튼** (우측 상단): 인벤토리 열기/닫기
- **제작대 버튼**: 제작대 열기/닫기
- **채팅 버튼**: 채팅창 열기/닫기
- **핫바 버튼** (하단 중앙): 슬롯 선택

## 설치 방법

### 요구사항

- Python 3.12 이상
- pygame >= 2.5.0
- Pillow >= 10.0.0

### 설치

```bash
pip install -r requirements.txt
```

## 실행 방법

```bash
python main.py
```

## 빌드 방법

Windows 실행 파일로 빌드:

```bash
python setup_cxfreeze.py build
```

또는

```bash
build.bat
```

## 게임 플레이

### 키보드 컨트롤 (데스크톱)

- **W/A/S/D**: 이동 및 점프
- **TAB**: 인벤토리 열기/닫기
- **T**: 채팅창 열기/닫기
- **마우스 좌클릭**: 블록 채굴
- **마우스 우클릭**: 블록 설치
- **마우스 휠**: 핫바 슬롯 변경

### 모바일 컨트롤

모든 기능이 터치 버튼으로 제공됩니다. 화면에 표시된 버튼을 터치하여 게임을 플레이하세요.

## 개발 정보
