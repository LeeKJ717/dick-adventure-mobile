# 리소스 디렉토리

이 디렉토리는 안드로이드 앱 빌드에 필요한 리소스 파일을 저장합니다.

## 필요한 파일

### 1. 아이콘 (icon.png)
- **크기**: 512x512 픽셀
- **형식**: PNG
- **요구사항**: 
  - 투명 배경 없음
  - 구글 플레이 스토어 정책 준수
  - 고해상도

### 2. 스플래시 스크린 (presplash.png)
- **크기**: 권장 1920x1080 또는 기기 해상도에 맞춤
- **형식**: PNG
- **용도**: 앱 시작 시 표시되는 화면

## 파일 생성 방법

### 온라인 도구 사용
1. [Canva](https://www.canva.com/) - 무료 디자인 도구
2. [GIMP](https://www.gimp.org/) - 오픈소스 이미지 편집기
3. [Photoshop](https://www.adobe.com/products/photoshop.html) - 전문 디자인 도구

### 아이콘 디자인 팁
- 간단하고 명확한 디자인
- 작은 크기에서도 인식 가능
- 게임의 핵심 요소 반영
- 브랜드 색상 사용

### 스플래시 스크린 디자인 팁
- 게임 로고 포함
- 로딩 인디케이터 (선택사항)
- 브랜드 일관성 유지

## 파일 배치

파일을 생성한 후 이 디렉토리에 저장하세요:
- `data/icon.png` - 앱 아이콘
- `data/presplash.png` - 스플래시 스크린

`buildozer.spec` 파일에서 이 경로를 참조합니다.
